# Salem Cyber Vault

Advanced cybersecurity monitoring dashboard with blue theme and floating eyes background effect.

## Features

- Real-time threat monitoring dashboard
- Interactive cyber search interface
- Advanced API features consolidated in one streamlined area
- Blue color palette theme
- Floating eyes background animation
- Ready for Vercel deployment at salemcybervault.com

## Deployment

This project is configured for deployment on Vercel with custom domain salemcybervault.com.

### Local Development

```bash
cd cybersecurity-dashboard-2
npm install --legacy-peer-deps
npm run dev
```

### Production Build

```bash
npm run build
```

## Technology Stack

- Next.js 15
- React 19
- Tailwind CSS
- TypeScript
- Framer Motion for animations
